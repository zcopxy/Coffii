# Coffii TikTok Carousels

Seven Warm Editorial carousel sets for promoting Coffii, generated from the shared art-direction brief.

## Exports

- `4x5/` contains 26 final slides at `1080 x 1350`.
- `9x16/` contains 26 final slides at `1080 x 1920`, with top and bottom safe margins considered for TikTok UI.
- `source/` contains the 26 generated clean photographic plates before typography and app-card overlays.
- `contact-sheets/` contains QA overview images for reviewing each carousel quickly.
- `manifest.json` lists every slide, overlay copy, source plate, and export path.
- `scripts/render_carousels.py` regenerates all exports from `source/`.

## Art Direction

The set uses the requested palette: Cream `#F6EFE5`, Paper `#FFFDF8`, Ink `#2C2018`, Espresso `#4A352E`, InkSoft `#5A4A3C`, Crema `#B8732E`, Crema soft `#E7C79A`, and hairlines `#E6DCCD`.

Typography is added in post with New York for headlines and Avenir Next for utility text so the copy stays exact.
